    ✨ প্রজেক্ট English Janala 
Enlish Janala হলো ইংলিশ টু বাংলা শব্দার্থ শেখার ওয়েবসাইট। এই ওয়েব এ শব্দের অর্থ , উচ্চারণ চর্চা করা যায়। এ ছাড়াও রয়েছে শব্দের সমার্থক শব্দ, শব্দ দিয়ে তৈরি বাক্য ইত্যাদি।

ওয়েবসাইট টি মূলত API এর মাধ্যমে ডাটা গ্রহন করে। 

     ওয়েবসাইটটি যেভাবে তৈরি করা: 

    ১। সর্বপ্রথম html ও tailwind css দিয়ে মোটামুটি ডিজাইন তৈরি করা হয়েছে।     

    ২। এরপর জাভা স্ক্রিপ্ট এর সাহায্যে fetch ফাংশন ব্যবহার করে API এর মাধ্যমে সার্ভার থেকে সকল লেসনস্ এর ডাটা গুলো নেওয়া হয়েছে।
         
    ৩। ইউজার যেকোনো লেসন ক্লিক করলে উক্ত লেসনের জন্য সকল শব্দের কার্ড গুলো ওয়েবসাইটে দেখানো হয়েছে।     

    ৪। প্রতিটি লেসনের কার্ডে একটি করে Info ও  Voice বাটন আছে ।    

    ৫। Info ক্লিক করলে ইউজার উক্ত শব্দের বিস্তারিত দেখতে পায় আবার Voice ক্লিক করলে উক্ত শব্দের উচ্চারণ শুনতে পায়- যেগুলো সম্পূর্ণ ডায়নামিক্যালি API এর মাধ্যমে করা হয়েছে।
